Titanic Survival Prediction using Multiple Machine Learning Algorithms

This project presents a comparative analysis of multiple supervised Machine Learning classification algorithms to predict passenger survival on the Titanic dataset.

The objective is to evaluate and compare different classification models to determine which algorithm performs best for binary survival prediction.

Algorithms Implemented

Logistic Regression

K-Nearest Neighbors (KNN)

Decision Tree

Naive Bayes

Support Vector Machine (SVM)

Project Workflow

Data loading and exploratory data analysis

Data cleaning and handling missing values

Feature engineering and categorical encoding

Train-Test split

Model training using multiple algorithms

Model performance evaluation and comparison

Technologies Used

Python

Pandas

NumPy

Seaborn

Scikit-learn

Problem Type

Binary Classification
Target Variable: Survived (0 = Not Survived, 1 = Survived)

Project Objective

To analyze and compare the performance of different machine learning classification algorithms and identify the most effective model for survival prediction.
